
```bash
# initialize (package) in current directory
$ cargo init

# create new (binary) package
$ cargo new <package-name>

# create new library package
$ cargo new --lib <library-name>

# build and run the code 
$ cargo run

# build code
$ cargo build

# build release version (i.e. apply optimizations)
$ cargo build --release

# run tests
$ cargo test

# remove old builds (debug and release)
$ cargo clean

# remove a package
$ cargo uninstall <package name>
```
