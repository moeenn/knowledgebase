#### Linux commands cheat-sheet

```bash
# check size of a file (-h means human-readable)
$ du -h /path/to/file
```

```bash
# kill a program
$ killall -v --ignore-case
```

```bash
# gain full access to a file or directory
$ sudo chmod -R a+rq /path/to/file/or/directory
```
